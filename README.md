Projeto Fullstack criado durante a Academia Java da empresa Atos em parceria com a Universidade Franciscana - UFN.
Principais tecnologias utilizadas:
Java,
JavaScript, 
HTML5, 
CSS3, 
Spring Boot,
Spring Security, 
Swagger e 
JWT.
